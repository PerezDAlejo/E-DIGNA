# E-DIGNA - Analizador de Contraseñas con IA 🔐

Una aplicación web moderna que utiliza inteligencia artificial para analizar la seguridad de contraseñas y proporcionar recomendaciones de mejora.

## 🚀 Características

- **Análisis en tiempo real** de la fuerza de contraseñas
- **Inteligencia artificial** powered by OpenAI GPT-3.5
- **Análisis básico** con zxcvbn como respaldo
- **Interfaz responsive** para todos los dispositivos  
- **Modo oscuro/claro**
- **Sugerencias inteligentes** de mejora
- **Versiones mejoradas** de contraseñas

## 🛠️ Tecnologías

- **Frontend**: HTML5, CSS3, JavaScript vanilla
- **Backend**: Node.js, Express.js
- **IA**: OpenAI GPT-3.5 Turbo
- **Análisis**: zxcvbn library
- **Deploy**: Vercel

## 🔧 Instalación Local

1. Clona el repositorio:
```bash
git clone https://github.com/tu-usuario/e-digna-password-analyzer.git
cd e-digna-password-analyzer
```

2. Instala dependencias:
```bash
npm install
```

3. Configura variables de entorno:
```bash
# Crea un archivo .env
OPENAI_API_KEY=tu-api-key-aqui
```

4. Ejecuta la aplicación:
```bash
npm start
```

5. Abre tu navegador en `http://localhost:3000`

## 🌐 Deploy en Vercel

1. Haz fork de este repositorio
2. Conecta tu cuenta de Vercel con GitHub
3. Importa el proyecto en Vercel
4. Configura la variable de entorno `OPENAI_API_KEY`
5. ¡Deploy automático!

## 📋 Variables de Entorno
